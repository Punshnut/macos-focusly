# Focusly Configurações & Primeiros Passos

Este guia apresenta a janela de configurações do macOS e as dicas da primeira execução que ajudam o Focusly a manter as distrações longe.

## Abrindo as configurações do Focusly
- Inicie o Focusly e procure o ícone na barra de menus (Minimal Dot, Halo ou Equalizer, conforme o estilo atual).
- Clique no ícone e escolha `Preferências…`, ou pressione `⌘,` com o Focusly ativo para abrir a janela de Preferências.
- A janela tem rolagem; as seções vão dos ajustes de aparência até automação e idioma.

## Tour pelas configurações

### Aparência
- `Focus Preset`: Escolha Smart Blur, Warm, Colorful ou Monochrome. A troca atualiza todos os monitores na hora.
- `Status Bar Icon`: Alterne entre Minimal Dot, Halo e Equalizer para combinar com sua barra de menus.
- Dica: Combine um preset mais suave com o ícone que você enxerga rapidamente sem perder o foco.

### Desempenho de rastreamento de janelas
- Escolha Standard (60 Hz), Energy Saving (30 Hz) ou High Performance (90 Hz) para definir com que frequência o Focusly acompanha o movimento das janelas.
- A legenda explica como o perfil escolhido afeta a resposta e o consumo de energia.
- Dica: Standard é o equilíbrio ideal; mude para High Performance quando reorganizar janelas com frequência.

### Monitores
- Cada tela detectada aparece como um chip. Clique em um chip para ajustar aquele monitor.
- Use `Excluir este monitor` quando quiser que o Focusly ignore totalmente uma tela.
- **Detalhes do desfoque**: Arraste o controle `Blur Style` para testar materiais do macOS — HUD Window, Menu, Popover, Sidebar, Sheet, Full Screen ou Window Background — e achar o que combina com o seu ambiente.
- **Efeitos de cor**: Ajuste `Overlay Strength`, modifique `Overlay Tint` e alterne entre `Preserve Color` e `Monochrome`.
- **Configurações padrão**: `Reset to Preset` restaura o monitor atual; `Apply to Other Displays` replica o visual nos demais monitores.
- Dica: Reduza a opacidade em telas secundárias para manter referências visíveis sem chamar tanta atenção.

### Atalhos
- Ative **Enable Focus Toggle Shortcut** para habilitar o atalho global.
- Pressione **Record** para gravar uma combinação (por exemplo, `⌃⌥⌘F`) e **Clear** para removê-la.
- Dica: Escolha uma combinação de fácil alcance com uma mão para mostrar a mesa rapidamente sem quebrar o fluxo.

### Iniciar ao fazer login
- Ative **Launch Focusly at login** para que as sobreposições iniciem junto com o macOS. Se o sistema bloquear a opção, o Focusly mostra instruções em Ajustes do Sistema → Usuários e Grupos → Itens de início.
- Dica: Iniciar ao fazer login mantém as sobreposições consistentes após reinicializações e ao conectar monitores externos.

### Idioma do app
- Use **App Language** para substituir o padrão do macOS, alternar instantaneamente entre traduções e revisar contribuições.
- Dica: Deixe em “Follow macOS Language (Default)” se quiser acompanhar o idioma definido no sistema.

### Rever a introdução
- Clique em `Rever introdução…` para reabrir o tour de onboarding quando quiser.
- Você também pode acessá-lo pelo ícone da barra de menus (`Mostrar introdução…`).

## Primeiros passos para trabalhar focado
1. **Ative as sobreposições**: No menu rápido, ligue `Ativar sobreposições` em cada monitor para suavizar o fundo.
2. **Escolha um preset por ambiente**: Smart Blur equilibra foco neutro, Warm combina com a noite, Colorful realça apresentações e Monochrome deixa monitores de referência discretos.
3. **Ajuste cada monitor**: Reduza a intensidade nas telas com mensagens e aumente o desfoque na principal para esconder papéis de parede agitados.
4. **Configure seus controles**: Grave um atalho global para mostrar a mesa quando precisar e ative “Iniciar ao fazer login” para manter o Focusly sempre pronto.
5. **Revise após mudanças**: Sempre que reorganizar monitores ou trocar presets, reabra a introdução pelo ícone da barra de menus para reforçar os gestos mais rápidos.
